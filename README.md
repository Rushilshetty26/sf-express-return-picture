# 顺丰电商退货地址截图模拟
- 简介  
通过退货渠道享受寄件优惠价，1KG内优惠力度大，且可叠加优化券(PS:比学生渠道还低)  
纯HTML，不依赖JS，保证隐私  
一个模版，多平台通杀：JD/STO/ZTO
- 使用方式  
Android: 下载模版文件-浏览器浏览文件-填入收件信息-截图-对应小程序上传即可  
IOS: 下载模版文件-使用"文件"app进行快速查看-填入收件信息-截图-对应小程序上传即可
